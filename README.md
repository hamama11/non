## 🗺️ 나만의 논술 로드맵, 이렇게 완성됩니다

| 단계 | 주요 기능 | 설명 및 활용 팁 |
|:---:|---|---|
| **📋 STEP 1**<br/>관심 대학 선택 | **일정표 연동 & 행열 전환** | • 일정표의 대학명을 한 번 클릭하여 시간표에 즉시 추가<br/>• **⇄ 행열 전환**으로 날짜/대학 기준 레이아웃 전환<br/>• 여대(분홍) · 서울(검정) · 서울 외(초록) 색상 범례 제공 |
| **🎯 STEP 2**<br/>맞춤 대학 추가 | **정밀 필터 & 직접 검색** | • **수리논술 범위 필터**: 미적분/확통/기하 포함·제외·무관 선택<br/>• **언어논술 필터**: 답안 분량(장/중/단문), 통계/수리/영어 제시문 여부 선택<br/>• **약술형 필터**: 약술형 전형만 모아보기 & PDF 자료 제공<br/>• **검색창**: 대학명으로 빠르게 찾아 직접 추가 |
| **📅 STEP 3**<br/>일정 & 정보 확인 | **모의 계획표 & 요약표** | • **계획표(달력)**: ON/OFF 토글로 중복 일정 조정, `＋ 새 시간표`로 다중 시나리오(Plan A/B/C) 비교<br/>• **요약표**: 시험 일시(시간·소요시간), 논술 유형(출제범위), 모집인원, 전형 방법 실시간 요약 |
| **↕️ STEP 4**<br/>지원 순서 조정 | **드래그 앤 드롭 정렬** | • 요약표의 각 행을 **드래그(↕)하거나 클릭**하여 원하는 우선순위(선호도, 합격 가능성 등)로 대학 순서 재배치 |
| **📷 STEP 5**<br/>완성본 저장 | **고해상도 이미지 캡처** | • 📷 **가로 캡처**: 시간표 + 요약표를 좌우 배치한 와이드 이미지 (화면 공유용)<br/>• 📸 **세로 캡처**: 시간표 위 + 요약표 아래로 이어지는 세로 이미지 (인쇄, 모바일용) |
