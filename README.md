## ✨ 핵심 기능 한눈에 보기

### 🔍 1. 대학 검색 & 달력 시각화
목표 대학을 검색해서 담으면, 우측 시간표에 **시험 일정이 달력 형태**로 즉시 펼쳐집니다.  
날짜·시간이 겹치는지 **팩트 체크**가 한눈에 가능합니다.

> 💡 일정표 대학명을 **클릭 한 번**으로 바로 추가할 수도 있어요!

---

### 🎯 2. 나에게 맞는 대학 추천 필터 (핵심!)

나의 강점·약점에 맞게 세밀하게 필터링하면, **찰떡같이 유리한 대학 리스트**를 자동으로 추려줍니다.

| 필터 구분 | 세부 항목 |
|---|---|
| 📐 **수리 필터** (자연/의약 전용) | 미적분 · 확률과 통계 · 기하 포함/제외 선택 |
| 📝 **언어 필터** (인문 전용) | 답안 분량(장문/중문/단문/자유형) · 제시문 유형(통계/수리/영어) |
| 🟣 **약술형 전형** | 약술형 대학만 모아보기 · 관련 PDF 다운로드 |

---

### 🗂️ 3. 멀티 시간표 탭 (Plan A / B / C 비교)

수시 6장, 경우의 수가 많죠? `＋ 새 시간표` 버튼을 눌러 **상향·적정·하향** 각 전략별 시간표를 **탭으로 생성**하고 나란히 비교할 수 있습니다.

---

### 📷 4. 가로/세로 고해상도 이미지 캡처

완성된 계획표를 **PNG 이미지**로 다운로드해서 친구·선생님과 바로 공유하세요.

| 캡처 유형 | 설명 |
|---|---|
| 📷 **가로 캡처** | 시간표와 요약표를 좌우로 나란히 배치한 **와이드 이미지** |
| 📸 **세로 캡처** | 시간표 아래 요약표를 세로로 나열한 **고해상도 길쭉 이미지** |

---

### 📊 5. 모의 지원 대학 요약 & 순서 조정

**ON 상태** 대학들이 하단 요약표에 자동 집계됩니다.  
행을 **드래그하거나 클릭**해 지원 우선순위 순서를 자유롭게 재배치할 수 있습니다.

## 🍯 수험생 꿀팁 3가지

#### 💡 꿀 1 — 📄 백분위별 지원 가능 대학 안내
내 모의고사 성적(백분위)으로 어느 대학 논술이 **상향·적정·하향** 라인인지 가이드라인을 확인하세요.

#### 💡 꿀 2 — 🥠 포춘쿠키 열기
수험 스트레스가 쌓였을 때 포춘쿠키를 까보세요. 따뜻한 응원 메시지가 기다리고 있어요.

#### 💡 꿀 3 — 📅 2027학년도 논술 일정표 한눈에 보기
서울시교육청에서 인정받은 **한 장짜리 전체 요약본**으로 큰 그림을 먼저 그려보세요.

## 🚀 3단계 사용법

```
STEP 1  ──  대학 탐색 및 담기
            방법 A: 달력 일정표에서 대학명 클릭 → 즉시 추가
            방법 B: 맞춤 필터(수리/언어/약술형) 설정 → 추천 목록 확인 → 추가
            방법 C: 검색창에 직접 대학명 입력 후 추가
                                    ▼
STEP 2  ──  계획표 수정 & 우선순위 정렬
            ON/OFF 토글: 시간표 반영 여부 실시간 전환
            순서 드래그: 요약 카드 끌어서 우선순위 재배치
                                    ▼
STEP 3  ──  최종 저장 & 공유
            가로 캡처(📷) / 세로 캡처(📸) PNG 이미지 다운로드
            새 시간표 탭으로 Plan A / B / C 시나리오 비교
```
